# Элиза - Голосовой Ассистент

Голосовое приложение-ассистент с личностью Элизы - духа медиума из викторианской эпохи.

## Описание

Приложение предоставляет голосовой интерфейс для общения с AI-ассистентом. Элиза говорит спокойно, глубоко и загадочно, сохраняя свой мистический характер.

### Возможности

- Голосовой ввод через микрофон
- Синтез речи через ElevenLabs
- Интеграция с Polza.ai для обработки запросов
- Выбор из 3 женских голосов
- Настройка системного промпта
- Регулировка температуры нейросети
- Адаптивный дизайн для мобильных устройств
- Загадочный и мрачный UI

## Установка

```bash
npm install
```

## Настройка

1. Получите API ключ от [Polza.ai](https://polza.ai)
2. Получите API ключ от [ElevenLabs](https://elevenlabs.io)
3. Откройте приложение и перейдите в настройки (кнопка с шестеренкой)
4. Введите оба API ключа в соответствующие поля
5. При желании настройте системный промпт и параметры

## Использование

1. Нажмите на большую круглую кнопку в центре экрана
2. Говорите в микрофон (кнопка покажет "Слушаю")
3. Отпустите кнопку или нажмите снова для остановки записи
4. Дождитесь обработки и ответа Элизы (кнопка будет пульсировать с надписью "Отвечаю")
